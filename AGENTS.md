# AGENTS

This file helps AI coding agents become productive quickly in this repository.

## Scope

- Primary package: parser library in src/
- Secondary package: docs site in docs/ (VitePress)
- Workspace uses pnpm workspaces (root + docs)

## Quick Start Commands

Run from repository root:

- Install: pnpm install
- Build library: pnpm build
- Test: pnpm test
- Lint: pnpm lint
- Format: pnpm format
- Typecheck: pnpm typecheck
- Full release checks: pnpm run release:check
- Run docs locally: pnpm docs:dev
- Build docs: pnpm docs:build

## Tooling Conventions

- Package manager: pnpm only (packageManager is pinned)
- Lint/format stack is oxlint + oxfmt (not ESLint + Prettier for repo linting)
- Typecheck command uses tsgo --noEmit
- Build uses tsdown and outputs dist/ from src/index.ts
- Module system is ESM (type: module)

## Project Map

- src/tokenizer: low-level tokenization and source tracking
- src/parser: parser API, errors, ESLint-facing parse flow
- src/constructor: AST construction from tokens via handlers
- src/types: AST and parser/tokenizer types
- src/constants: enums and parser/tokenizer constants
- src/utils: AST helpers (traversal, search, clone, validation)
- tests/parse: parser behavior coverage for tags, attrs, comments, xml, edge cases

Public API entrypoint is src/index.ts.

## Agent Workflow Expectations

- Prefer minimal, targeted edits; avoid broad refactors unless requested
- Add or update tests when behavior changes
- Run at least: pnpm test and pnpm typecheck after non-trivial parser changes
- For release-quality changes, run pnpm run release:check
- If changing docs-facing behavior or API, update docs in docs/api or docs/guide

## Gotchas

- Project is marked WIP; API can change between versions
- docs/components.d.ts is generated by unplugin-vue-components; do not hand-edit
- docs package depends on local parser via workspace:\* and should be run through root pnpm workspace
- tsdown config marks eslint as neverBundle; preserve that behavior unless intentionally changing distribution strategy
